# nw-obs-production

Nuclear Winter Club — OBS production assets and configuration.

## Folder Structure

- `html/` — Browser source HTML files (overlays, countdowns, etc.)
- `obs-config/` — Exported OBS scene collections and profiles
- `scripts/` — OBS scripts
- `assets/` — Media files, videos, images

## Setup

Import scene collections via OBS: *Scene Collection → Import* and select the relevant JSON from `obs-config/`.
